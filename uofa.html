Style guide application

I custom built a style guide application which contained all the HTML, CSS and JavaScript used by all development teams accross the university for developing public website and applications, and internal systems. The style guide application allowed development teams to simply reference a minified version of the assets, or to custom build their own by pulling components and templates from the git reporitoty.
